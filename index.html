Introduction

Hello reader! You've stumbled across Daniel's hypertext page. This site is where I will, or, where I hope to, post stuff about various things.



Why create a website?

The whole reason why I created this website is because I am currently learning HTML. For those of you who don't know, HTML is a programming language used to organize and present raw text. Thus, it is the coding language used for designing websites (along with CSS, Javascript, etc). This entire website will be hand-coded by me. That means all of the fonts, colors, locations of words and pictures on your screen were not dragged and dropped and selected using a website editor. All of those little features had to coded by hand by me. Now, by no means am I saying that HTML is hard. Actually, it's probably the easiest programming language that one can learn. I guess what I'm trying to say is that coding a website by hand is a lot more work. So, I'm really just trying to show off.
As of now, I am mostly focused on adding to the functionality of the website: that means adding new pages, adding content, and updating information. At some point, I will create a roadmap so that you all will be able to see the plans for the future.
